How It Works

1

Intercept

Agent Firewall sits as a transparent TCP proxy between your AI agent and the MCP server, intercepting every tool call before it touches the OS.

2

Evaluate

Each call is matched against a YAML policy — hard-block patterns, credential regexes, reverse-shell signatures, and privilege escalation commands.

3

Decide

Safe operations pass through silently. Dangerous ones are blocked immediately. Borderline cases show a native popup — you decide with full context.
